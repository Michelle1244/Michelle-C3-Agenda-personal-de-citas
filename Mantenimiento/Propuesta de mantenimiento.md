# ***Mantenimiento Propuesto para el Sistema de Agenda personal de citas***

## ***1. Problema: Ocasionalmente, el sistema no logra sincronizar correctamente las citas entre los distintos dispositivos, generando discrepancias y evitando que ciertos cambios se reflejen de manera inmediata.***



  El sistema al estar mal optimizado dificulta la sincronizacion entre varios equipos siendo algo molesto para el usuario



  ***Tipo de Mantenimiento:*** Mantenimiento correctivo



  ***Acción Principal:*** Evaluar el funcionamiento del módulo de sincronización y aplicar las correcciones necesarias para optimizar el intercambio de datos, garantizando una actualización adecuada en tiempo real.



  ***Justificación Técnica:*** Atender este inconveniente es esencial para preservar la consistencia de la información y asegurar que el usuario cuente con sus citas correctamente sincronizadas en todos sus dispositivos.



## ***2. Problema: La aplicación no ofrece opciones avanzadas de filtrado para buscar citas antiguas, lo que dificulta al usuario encontrar información específica rápidamente.***



  El sistema no realiza los filtros de busqueda adecuados lo cual es importante para que el usuario realice sus busquedas rapidamente



  ***Tipo de Mantenimiento:*** Mantenimiento Perfectivo



  ***Acción Principal:*** Implementar nuevos filtros como búsqueda por palabra clave, categoría, estado de la cita y fecha exacta.



  ***Justificación Técnica:*** EEstas mejoras incrementan la usabilidad y permiten acceder a la información de manera más eficiente, optimizando la experiencia del usuario.
  
  El sistema deberia ser mas optimizado y util para los usuarios
